HARLEM (HAamiltonians for Research of LargE Molecules)

HARLEM is a modular molecular modeling and analysis program with a cross-platform GUI. It is interfaced to major molecular modeling engines, such as AMBER, GROMACS and Gaussian.

HARLEM recognizes a range of molecular formats: PDB, XYZ, MOL2, HIN

Python is used for scripting and extensibility.

HARLEM is available for UNIX and WINDOWS operating systems.

The program is distributed under the open software license.

Source code of HARLEM is hosted at GITLAB
